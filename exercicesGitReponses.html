Pour renommer un fichier en utilisant la commande git mv, exécutez la commande git mv suivie du nom du fichier existant et du nouveau nom de fichier: git mv mon_fichier.txt mon_nouveau_fichier.txt git commit -m "Renommage du fichier mon_fichier.txt en mon_nouveau_fichier.txt" Pour cloner un dépôt Git distant en utilisant la commande git clone, exécutez la commande git clone suivie de l'URL du dépôt pull: git pull Pour envoyer les modifications locales au dépôt distant en utilisant la commande git push, exécutez la commande git push suivie du nom de la branche à pousser: perl git push origin master Pour annuler la dernière modification en utilisant la commande git revert, exécutez la commande git revert HEAD pour annuler le dernier commit: git revert HEAD Pour ignorer un fichier en créant
